Join Infernorifts

Quick instructions for Java, Bedrock, and Console players so you can jump into the Inferno with minimal fuss.

Java Edition

Desktop (Java) — Minecraft 1.21.8 and Above

IP
play.infernorifts.com
  1. Open Minecraft → Multiplayer → Add Server
  2. Enter the Server Address: play.infernorifts.com
  3. Click Join — enjoy the server!

Bedrock (PC / Mobile)

Windows 10, iOS/Android, and Bedrock-compatible clients

IP
play.infernorifts.com
Port
25565
  1. Open Minecraft (Bedrock) → Play → Servers → Add Server
  2. Enter play.infernorifts.com with port 25565
  3. Save & Join

Console Players

If you're on Xbox, PlayStation or other console — use our Bedrock Realm gateway.

Realm Code
YtF2cfjBwBJRkSE
  1. Open Minecraft on your console → Realm join
  2. Enter the Realm Code above → Join the gateway

Helpful notes

  • Make sure you're running Minecraft 1.21.8 (Java). For Bedrock clients, keep the client up-to-date for compatibility.
  • Read our Server Rules before you play — being familiar with the rules avoids accidental bans.
  • If you can't connect, try restarting Minecraft and your router. Still stuck? Check the FAQ or contact staff via the Contact page.
